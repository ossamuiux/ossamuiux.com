자기주도(SDL) Self-Directed Learning

토크빈의 대표적인 자기주도 학습 스스로 공부하고 싶은 주제를 선택하여, 선생님과 상의하에 수업!
관심있는 주제로 수업하기 때문에, 몰입도와 학습효과 극대화!

레벨 : BC~ 수강연령 : 6세~

4D Intensive Speaking

토크빈이 개발한 말하기 집중 수업, 다양한 주제를 통하여 어휘력, 문법 등을 말하기에 구현 레벨에 따라
Drill -> Description -> Debate 방식으로 진행 여러가지 학습방식으로 학습효과 극대화!
토크빈의 커리큘럼을 따라 안정적인 수업이 가능합니다.

레벨 : PC~ 수강연령 : 11세~

Essay Writing

미국 고등학교, 대학교 수준에서 요구되는 에세이 작문 수업, 에세이의 기본 Structure와 개념을 배우고 실제 에세이를 작성하며이해!
1개 주제에 따라 6회간 체계적인 수업을 진행하여 논문수준의 에세이 도출!

레벨 : BC~IA 수강연령 : 7세~

4D Speaking Grammar

우리나라의 문법수업은 시험위주의 외우기로 되어있어 특히 초등학생은 이해하기도 어렵고, 영어를 더 지루하고 어렵게 만듭니다.
토크빈 스피킹 그래머는 미국과 같은 방식으로 문법을 자연스럽게 익히도록 구성되었습니다.
문장을 만들면서, 단 복수, 3인칭 시제, 과거/미래 시제, 여러 문장형, 주어/동사 등 문장구성, 복잡한 표현 등으로 레벨에 따라 진행됩니다.

레벨 : BC~PC 수강연령 : 7세~

Book Discussion

재미있는 책을 원서로 읽고, 선생님과 이야기하고, 토론하는 클래스
리딩, 논리적 사고, 스피킹의 이상적인 조합
현재 제공 클래스 BC:매직트리하우스 PA~PB:웜피키드 PC~:해리포터

레벨 : PC~ 수강연령 : 11세~

디베이트(토론)

영어실력이 뛰어난 학생을 대상으로 진행
논리적 사고(logical thinking)을 기반으로 사고의 체계화, 논리 설계진행
실제 토론 기술 교육 및 연습, 실전 수업

레벨 : PB~ 수강연령 : 11~15세
